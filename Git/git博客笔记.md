## 一、Git的作用

- 版本控制，更新迭代
- 协同开发（branch）

## 二、常见的版本控制工具

- Git（分布式版本控制）
- SVN（集中式版本控制）
- CVS
- VSS

## 三、版本控制的分类

- 本地版本控制
- 集中式版本控制
- 分布式版本控制

## 四、git的全局配置

1. git config -l
   - git config --system --list
   - git config --global --list
2. 配置用户名和邮箱
   - git config --global user.name "yourname"
   - git config --global user.email "yourmail@mail.com"
